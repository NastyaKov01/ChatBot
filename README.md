# ChatBot

Чат-бот в Telegram, который дает рекомендации по выбору подарка на день рождения: https://t.me/presents_advisor_bot .

Для начала работы с ним надо ввести команду /start .
Затем бот попросит указать, кому нужно подобрать подарок, ввести пол, возраст и увлечения именинника.

**Requirements:**

1. aiogram

2. python-dotenv

3. surrogates
